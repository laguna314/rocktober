# Rocktober Fest Landing (React + Vite + Tailwind)

## Quick Start
```
npm i
npm run dev
```

## Build
```
npm run build
npm run preview
```

## Cloudflare Pages
- Build command: `npm run build`
- Output dir: `dist`

## Assets
- `public/rocktober.png` included (your background)
- Add `public/poster.jpg` for the hero poster or change the path in `src/App.jsx`.
